# 深层内容编辑第一版样本库

## 目的

本样本库把真实课件中与当前 GOAL 有关的结构提炼成可重复、可提交、无真实学生数据的最小 HTML。原课件始终只读，不复制进仓库。

样本仅用于发现和验收：它不表示当前功能已经实现，也不把样本中的 B 类互动、Canvas 或 iframe 内部编辑拉入第一版。

## 来源与脱敏原则

- 来源为桌面课件评审目录中的 G1–G4、G7–G12 课件。
- 仓库只记录课程编号，不记录原始绝对路径。
- 不复制真实令牌、接口地址、学生输入、完整教案或大段视觉素材。
- 外部 iframe 使用 `example.invalid`，运行时预览只使用 `srcdoc`。
- AI 生成样本只保留离线回退，不发起网络请求。

机器可读清单位于 `test/fixtures/deep-content-v1/manifest.json`。

## 覆盖边界

样本库必须保持 5–8 个样本，并持续覆盖：

- 弹窗与嵌套弹窗。
- 点击后出现的普通 DOM。
- 同源子 HTML 正常入口和损坏入口。
- 原课件脚本与表单状态保留。
- 外部 iframe 与运行时 iframe 两种反例。

未来增加样本必须先说明它补充了哪个真实缺口；不能为了数量扩大第一版范围。
