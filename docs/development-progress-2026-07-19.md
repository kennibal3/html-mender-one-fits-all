### 2. B1–B8 范围升级

更新 `docs/GOAL-互动功能完整验收.md`，将 B4–B8 从后续清单提升为正式范围。

已确认：

- B4 只使用可视化安全公式，不开放任意 JavaScript。
- B5 支持分值、部分得分、尝试扣分、总分、进度和解锁。
- B6 同一学习者同一次练习保持稳定，重新练习后才更新随机结果。
- B7 每个计时器单独配置离开暂停或继续。
- B8 使用媒体互动时间轴，并提供轻量媒体编辑。
- 轻量媒体编辑包含裁剪、分段、排序、音量、淡入淡出、封面、字幕和简单画面裁切。
- 不建设专业多轨、绿幕、复杂特效和专业关键帧。

### 3. 最小场景模型

新增 `src/scene-model.js`：

- 场景清单版本为 1.0。
- 为每个页面建立稳定根场景。
- 从 A2 弹窗互动生成弹窗场景。
- 自动推导嵌套弹窗父子关系。
- 忽略损坏互动清单但保留页面。
- 防止循环父子关系。

服务器已经在导入、保存、页面管理和重开时维护场景清单。

### 4. 活动节点与多层弹窗

修改两条运行路径：

- `vendor/html-slide-mender/assets/html-slide-mender-runtime.js`
- `vendor/html-slide-mender/assets/html-slide-mender-interactions.js`

完成：

- 不再克隆弹窗目标。
- 保留真实节点和内部 `data-hsm-node-id`。
- 保留原事件监听、输入值和运行状态。
- 使用占位节点恢复原位置。
- 使用弹窗栈支持父子层级。
- 子层关闭后恢复父层状态。
- Esc、遮罩和关闭按钮只关闭当前最上层。
- 编辑预览与导出运行派发相同场景进入、退出事件。

## 九、当前已知技术风险

- 编辑预览把活动节点放入编辑器 Shadow DOM，事件和状态已经验证，但复杂页面 CSS 需要继续做真实视觉回归。
- 导出弹窗会改变目标节点的祖先结构，依赖复杂祖先选择器的页面可能需要兼容处理。
- 极端不规范 HTML 可能超出轻量场景扫描器能力。
- SPA、动态 DOM、跨域 iframe 和封闭组件仍未进入实现阶段。
- 当前尚无场景返回工具条，不能把底部“退出编辑并返回列表”当作返回上一层。
